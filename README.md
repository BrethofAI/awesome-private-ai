This list will catalog AI tools — local, self-hosted, or hosted — that treat your data as yours. Privacy-respecting by architecture, not just by policy page. Different filter from [awesome-local-ai](https://github.com/BrethofAI/awesome-local-ai): private ≠ local. A self-hosted cloud service can be private. A local tool that phones home isn't.

If you maintain or use a tool that fits, [open an issue](../../issues/new)
to nominate it. Your suggestion will land in the first proper release.
